*Effective: March 1, 2026*

We collect minimal data to understand how this documentation is used. We have no user accounts, no cookies, no forms, and no local storage.

## Cloudflare Web Analytics

We use [Cloudflare Web Analytics](https://www.cloudflare.com/web-analytics/) to measure page views, visitor location, browser type, and page load performance. This is **cookieless** — no tracking cookies are set. Cloudflare may process your IP address to determine approximate location, but we do not have access to individual IP addresses. Data is retained by Cloudflare for 7 days.

See [Cloudflare's Privacy Policy](https://www.cloudflare.com/privacypolicy/) for details.

## GitHub Pages

This site is hosted on [GitHub Pages](https://pages.github.com/). GitHub may log visitor IP addresses for security and operational purposes.

See [GitHub's Privacy Statement](https://docs.github.com/en/site-policy/privacy-policies/github-general-privacy-statement) for details.

## GitHub API

We make a client-side request to the GitHub API to display the repository star count. This request is made by your browser directly to GitHub — we do not proxy or store any data from it.

## Changes

We may update this policy as our tooling changes. The latest version is always available at this URL.
